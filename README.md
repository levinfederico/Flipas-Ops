# Flipas-Ops — Agente de oportunidades

Scraper automático de PHs y deptos en CABA para flipping.

## Portales: MercadoLibre · Zonaprop · Argenprop · Remax
## Umbrales: 🔴 <$1.300 · 🟠 $1.300–1.400 · 🟡 $1.400–1.500

## Setup rápido
1. Completar `config.py` con IDs de Sheets
2. Agregar Secrets en GitHub (SMTP_PASSWORD, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID)
3. Activar GitHub Pages → Branch: main
4. GitHub Actions corre cada 2hs automáticamente
